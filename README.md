# ECommerce_Sales_Analysis
##ECommerce_Sales_Analysis_Dashboard

##Project Overview

This is an END-TO-END Power BI project developed to analyze the ECommerce sales performance.The dashboard provides a comprehensive view of business health by tracking key 
metrics,identifying sales trends,and understanding customer purchasing patterns.

#Project Type

This project was independently developed as a self-learning inititative to apply data cleaning,analysis,and dashboard creation skills using Power BI.

##Objective

The primary goal of this project was to transform raw ecommerce data into an interactive visual tool.This helps the business identify profitable regions,top-selling product
categories,and peak sales months to drive strategic decision-making.

##Dataset Information

The dataset used in this project contains e-commerce transaction details including order date,product quantity,and region.
The data was cleaned and transformed using Power Query.Necessary data modeling and calculated measures were created to generate meaningful insights for the dashboard.


##Tools

-Microsoft Power BI(Data Visualization and Dashboard Creation)
-Power Query(Data Cleaning and Transformation)
-DAX(Measures and KPI Calculation)

##Key KPIs

-Total Sales
-Total Profit
-Total Quantity Sold
-Category-wise Sales performance 
-Region-wise Sales Distribution
-Monthly Sales Trend
-Top Performance Products

##Dashboard Features

-Interactive dashboard with slicers and filters
-Category-wise and region-wise sales analysis
-Monthly trend analysis of sales and profit
-Top products performance comparison 
-Clean and user-friendly dashboard layout

##Key Insights

Cash on Delivery and Online Payment are the most commonly used payment modes,contributing significantly to total sales.Profit shows monthly fluctuations,
indicating seasonal trends in sales performance.Certain states generate higher sales and profit compared to others highlighting strong regional performance.
A few key customers contribute a major portion of total revenue,showing high_value customer segments.
Category-wise analysis show that specific product categories drive maximum sales and profitability.

##Conclusion

This project provides a comprehensive analysis of e-commerce sales data using Power BI.The dashboard helps in understanding sales trends,product performance,and
regional distribution,enabling data-driven business decisions and improving overall business performance.
